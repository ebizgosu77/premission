# SK Networks AI캠프 — 사전미션 관리 플랫폼

개강 전 수강생들의 사전학습(Python, Git)과 수학 미션을 관리하는 웹앱입니다.
순수 Vanilla JavaScript + CSS + HTML로 구성되며, 외부 라이브러리 없이 동작합니다.

---

## 파일 구조

```
aicamp-premission/
├── index.html          ← 로그인 화면 (수강생/매니저 선택)
├── student.html        ← 수강생 대시보드
├── manager.html        ← 매니저 대시보드
├── css/
│   └── style.css       ← 전역 스타일 (딥 네이비 테마)
├── js/
│   ├── storage.js      ← localStorage CRUD 유틸리티
│   ├── app.js          ← 공통 로직 (진행률 계산, 토스트, 모달)
│   ├── student.js      ← 수강생 전용 (탭, 퀴즈, 수학 미션)
│   └── manager.js      ← 매니저 전용 (현황, 문제관리, 설정)
├── data/
│   └── missions.js     ← 미션 데이터 (챕터, 퀴즈 문항, YouTube 링크)
└── README.md           ← 이 파일
```

---

## 시작 방법

1. `aicamp-premission` 폴더를 다운로드합니다.
2. `index.html` 파일을 브라우저(Chrome 권장)로 엽니다.
3. 수강생 또는 매니저로 입장합니다.

> 별도의 서버 설치 없이 브라우저에서 바로 실행됩니다.

---

## 수강생 사용법

1. **로그인**: `index.html`에서 이름을 입력하고 "입장하기" 클릭
2. **학습하기**: Python/Git 탭에서 강의 영상 시청 후 "수강 완료" 체크, 퀴즈 풀기
3. **미션 제출**: 사전미션(기본/심화) 탭에서 답안 작성 → "임시저장" 또는 "최종 제출"

### 주요 기능
- Python 8챕터: 각 챕터별 YouTube 강의 + 퀴즈 3문항 (70점 이상 합격)
- Git 5챕터: YouTube 강의 수강 체크
- 수학 미션: 매니저가 등록한 문제에 대한 서술형 답안 제출
- 수강 완료 체크 후 퀴즈 활성화
- 퀴즈 제출 후 오답 해설 확인 및 재응시 가능
- 최종 제출된 수학 답안은 수정 불가

---

## 매니저 사용법

1. **로그인**: `index.html`에서 비밀번호 입력 (초기 비밀번호: `aicamp2025`)
2. **현황 확인**: "전체 현황" 페이지에서 수강생별 진행률 확인, 이름 클릭으로 상세 조회
3. **문제 관리**: "문제 관리" 페이지에서 기본/심화 수학 문제 텍스트 편집 → 저장

### 주요 기능
- 수강생별 Python/Git/수학 진행률 색상 코딩 (빨강→주황→노랑→초록)
- 진행률 낮은 수강생 상단 노출 (도움 필요한 학생 우선)
- 개인 상세 모달: 챕터별 수강/퀴즈 현황 + 제출된 퀴즈 답안 정답/오답 표시 + 수학 답안 전문
- 비밀번호 변경, 기수명 변경, 전체 데이터 초기화 (2단계 확인)

---

## 문제 등록 방법

1. 매니저로 로그인
2. 좌측 사이드바에서 "📝 문제 관리" 클릭
3. 기본수학 / 심화수학 텍스트에어리어에 문제 내용 입력
4. "저장하기" 클릭
5. 수강생 화면에 즉시 반영됩니다.

---

## 기수 교체 시 초기화 방법

1. 매니저로 로그인
2. "⚙️ 설정" 페이지 이동
3. "기수명 변경"에서 새 기수명 입력 (예: "2기")
4. "⚠️ 전체 데이터 초기화" 버튼 클릭 (2단계 확인 필요)
5. 모든 수강생 데이터, 문제, 설정이 삭제되고 초기 상태로 돌아갑니다.

---

## 주의사항

- **localStorage 기반**: 모든 데이터는 브라우저의 localStorage에 저장됩니다.
  - 같은 브라우저, 같은 기기에서만 데이터가 유지됩니다.
  - 브라우저 캐시/데이터를 삭제하면 모든 데이터가 사라집니다.
  - 시크릿/프라이빗 모드에서는 창을 닫으면 데이터가 사라집니다.
- **동시 접속**: 여러 탭에서 동시에 사용하면 데이터 충돌이 발생할 수 있습니다.
- **저장 용량**: localStorage 용량은 약 5~10MB입니다. 수강생이 많아지면 용량 부족 알림이 표시됩니다.
- **비밀번호**: 초기 매니저 비밀번호는 `aicamp2025`입니다. 첫 로그인 후 반드시 변경하세요.
